# Data Sources & Verification

All venues and properties in this map include direct links to their original sources so you can verify details and access official information.

## Global Dining Credit (2,439 restaurants across 16 countries)

**Source**: Amex Platinum Dining Partner Network  
**Coverage**: Australia, Austria, Canada, France, Germany, Hong Kong, Italy, Mexico, Monaco, New Zealand, Singapore, Spain, Taiwan, Thailand, United Kingdom, United States

**Verification**:
- Click any restaurant on the map
- In the details panel, look for **"Open in Google Maps"** to see ratings, reviews, hours, and user photos
- For Japan restaurants specifically, see the **Tabelog** link for authentic Japanese restaurant ratings

**What's included**:
- Restaurant name, address, cuisine
- Phone number (where available)
- Neighborhood/district
- Pricing bands (Japan: lunch/dinner price ranges in ¥)
- Reservation requirements
- Menu language support (Japan)
- Google Maps ratings & review counts

## How Data Was Collected

### Global Dining Credit
1. Extracted from `platinumdining.caffeinesoftware.com` partner network
2. Geocoded using JSON-LD structured data and Google Maps API
3. Deduplicated and validated for coordinate accuracy
4. Cross-checked against country geographic boundaries

### Plat Stay
1. Extracted from official AMEX Platinum Travel Benefits documentation
2. Geocoded using geographic coordinates from property records
3. Validated for data completeness and accuracy

### Love Dining
1. Extracted from official AMEX Singapore Platinum Dining Program
2. Geocoded with Singapore-specific coordinate validation
3. Verified to ensure all coordinates fall within Singapore bounds
